Alert tools manage notification configuration for NPA operations.

| Tool | Use it for |
|------|------------|
| `getAlertConfig` | Read current alert settings. |
| `updateAlertConfig` | Update alert recipients or event settings. |

## Safe prompts

```text
Show the current NPA alert configuration and identify who receives publisher offline and upgrade failure notifications. Do not change it.
```

## Change prompts

```text
Plan alert updates for the Tokyo IT team. Show current recipients, proposed recipients, and event types before applying changes.
```
