title: Steering Tools
date: 2026-05-20
---

Steering tools manage publisher associations for private applications.

| Tool | Use it for |
|------|------------|
| `addPublisherAssociation` | Add app-publisher association. |
| `updatePublisherAssociation` | Replace or update associations. |
| `deletePublisherAssociation` | Remove an association. |

## Safe prompts

```text
For the CRM private app, show which publishers are associated and whether there is regional redundancy. Do not change associations.
```

## Change prompts

```text
Plan a publisher failover change for the CRM app. Show current associations, proposed associations, and impact before using write tools.
```
